# Benchmark And Figure Assets

This directory stores lightweight benchmark summaries and figure files used to document TorchGWAS behavior.

## Tracked Content

- `benchmarks/`
  Tabular summaries used for reproducibility and figure generation
- `figures/`
  Derived visualization assets generated from tracked benchmark tables

The primary benchmark-generation workflow is documented in:

- [docs/reproduce-benchmarks.md](../docs/reproduce-benchmarks.md)
